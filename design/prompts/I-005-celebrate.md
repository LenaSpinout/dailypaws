# I-005 – Celebrate

Base style: see `design/prompts/base-style.md` — this prompt only
adds the scene-specific detail on top of that base.

Reference: `docs/IllustrationCatalogue.md` — I-005 Celebrate.

## Used On

Mission screen (Celebrate Missions).

## Purpose

Support Celebrate Missions.

## Emotion to Communicate

- Pride
- Joy
- Connection

## Scene

The default character and the default dog sharing a warm, joyful
moment together — close, relaxed, delighted in each other's company.
Per `docs/MissionFramework.md`'s Mission Types, Celebrate means
"enjoy a shared moment together," so the joy in this scene belongs to
the relationship, not to an accomplishment.

## Avoid

- Trophies, medals, badges, ribbons or any achievement/reward
  iconography — per `docs/MissionFramework.md`, a DailyPaws Mission
  is never a competition or a score, and Celebrate is no exception.
- Any framing that reads as "you did it" rather than "we enjoyed
  this."
