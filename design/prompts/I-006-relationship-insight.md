# I-006 – Relationship Insight

Base style: see `design/prompts/base-style.md` — this prompt only
adds the scene-specific detail on top of that base.

Reference: `docs/IllustrationCatalogue.md` — I-006 Relationship
Insight.

## Used On

Insight screen (Relationship Insights).

## Purpose

Support relationship insights.

## Emotion to Communicate

- Trust
- Reflection
- Togetherness

## Scene

The default character and the default dog in a quiet, still moment
together — sitting together, looking at each other, resting side by
side. Per `docs/IllustrationGuide.md`'s Insight Illustrations
principle, Insight illustrations should communicate reflection, using
moments like sitting together, looking at each other, resting or
quiet togetherness.

## Avoid

- Any reward, achievement or "shared success" framing —
  `docs/IllustrationGuide.md` is explicit that Insights should never
  feel like rewards or achievements.
- Action or activity — this illustration is about a pause for
  reflection, not doing something together.
