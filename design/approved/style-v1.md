# DailyPaws Illustration Style v1

Status: Approved for Explorer Release

## Selected Style

After evaluating several illustration directions, the project selected:

**Style 2 – Illustrated & Friendly**

This becomes the official illustration style for the Explorer Release.

## Why we selected it

The style best communicates the DailyPaws personality:

- warm
- friendly
- approachable
- emotionally engaging
- suitable for adults while remaining playful
- works well in both onboarding and the application

Compared to the more realistic style, it feels more welcoming.

Compared to the minimalist style, it communicates more personality and emotion.

## Notes

Illustrations should follow IllustrationGuide.md.

This decision applies to Explorer Release only.

The visual identity may evolve in future releases based on user feedback.

## Explorer Release Decision

For the Explorer Release (v0.1), all illustrations will use the same default character.

This is an intentional simplification to:

- keep the illustration library small,
- maintain visual consistency,
- reduce implementation effort,
- validate the illustration concept before investing in personalization.

The selected default character is:

- Adult woman
- Warm Scandinavian appearance
- Casual outdoor clothing
- Medium-sized brown doodle-like dog

## Future Direction

In future releases, illustrations should become personalized.

The illustration system should support multiple character variants, for example:

- woman
- man
- adult couple
- child
- family

Illustrations should eventually match the onboarding information whenever possible, while preserving the same illustration style and emotional tone.

This personalization is intentionally out of scope for the Explorer Release.
