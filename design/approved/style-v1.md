Status: Approved

DailyPaws-StyleBoard-v1.png

This Style Board takes precedence over any written description.

If a generated illustration differs from the Style Board, the Style Board always wins.

IllustrationGuide.md defines the principles.

The Style Board defines the visual language.

## Explorer Release implementation decision

For the Explorer Release, DailyPaws will temporarily use a warm, friendly, semi-realistic illustrated style that can be generated consistently with the current image-generation tool.

This is an implementation compromise, not a change to the long-term visual direction.

The approved DailyPaws Style Board remains the target visual identity for future releases.

After Explorer testing, the illustration library should be reviewed and may be regenerated or redrawn using a tool or illustrator better suited to the approved target style.
