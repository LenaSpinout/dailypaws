# Sprint 3 – A Week Together

## Goal

Make DailyPaws engaging enough that a family genuinely wants to return tomorrow.

## Success Criteria

- Mission library feels varied.
- Home feels alive.
- The journey feels continuous.
- Ready for one week of external family testing.

## Scope

Included

- #20 Mission Library
- #29 Home Status
- #30 Navigation + Profile
- #24 Timeline
- #25 Empty/Error States
- #26 PWA Install

Excluded

- #21 Insight Learning
- #22 Conversations
- #23 Conversation Storage
- #33 Conversation Memory

## Why

Sprint 3 is about improving the experience—not increasing intelligence.

The goal is to validate engagement before introducing adaptive behaviour.
