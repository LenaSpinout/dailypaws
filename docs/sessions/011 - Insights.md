# Session 011 – Insight

## Goal

Define the Insight concept.

---

## Decisions

Insights are meaningful interpretations rather than facts.

Insights require sufficient evidence before being presented.

Users can confirm, reject or refine every Insight.

Insights should always be constructive.

Four Insight types were defined:

- Dog
- Relationship
- Progress
- Discovery
